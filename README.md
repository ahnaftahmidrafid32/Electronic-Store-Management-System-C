# Electronic Store Management System (C)

A console-based Electronic Store Management System written in C.

## Features
- Product management (Add, View, Update, Delete, Sell)
- Customer management
- Supplier management
- Employee management
- File-based data storage
- Sorting and reporting system

## Technologies Used
- C Programming Language
- File Handling
- Structures
- Modular Functions
